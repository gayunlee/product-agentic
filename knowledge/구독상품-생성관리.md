# 구독상품 생성/관리 방법

출처: Slack Canvas `F0AKF6F79K6` (2026-03-09)

---

## STEP 1. 상품 페이지 생성하기

우측 상단 `페이지 생성` 버튼을 눌러서 새로운 상품 페이지를 생성합니다.

---

## STEP 2. 상품 페이지 정보 설정

상품의 노출 조건과 판매 방식을 설정하는 영역입니다.

| 순번 | 항목 | 설명 |
|------|------|------|
| 1 | **히든 처리 여부** | 메인 상품페이지(어스플러스 홈/구독 탭) 외에 특정 프로모션이나 개별 링크로만 접근하게 할 경우 '히든'으로 설정 |
| 2 | **마스터 선택 (오피셜클럽 선택)** | 상품을 판매할 오피셜클럽을 선택 |
| 3 | **페이지명** | 관리자센터에서 노출되는 명칭 (고객에게는 노출되지 않음) |
| 4 | **페이지 공개 여부** | **공개**: 설정된 '공개 기간'에 맞춰 실서비스에 노출. **비공개**: 페이지가 숨겨짐. 최종 검수 전에는 비공개 권장 |
| 5 | **공개 기간** | 상품 페이지에 접근 가능한 기간. 오늘 날짜가 기간 내에 포함되어야 페이지가 활성화 |
| 6 | **상품 타입** | **구독상품**: 정해진 주기(N개월)마다 정기적으로 결제. **단건상품**: 한 번만 결제 |
| 7 | **신청 기간** | '구매하기' 버튼이 동작하는 판매 기간. 종료되면 '모집이 종료된 멤버십입니다' 표시 |
| 8 | **상품 변경 가능 여부** | 구매 고객이 직접 상품 옵션을 변경할 수 있게 허용할지 선택 |
| 9 | **편지글 공개 여부** | '공개'로 설정 시 상품 페이지 내 [편지글] 탭이 활성화 |

## STEP 3. 상품 페이지 이미지 설정

상품 페이지에 이미지/비디오 자료를 등록합니다. 생성 이후에도 수정 가능.

### 사용자 화면 구성

| 순번 | 항목 | 설명 |
|------|------|------|
| 1 | **상품 대표 이미지/비디오** | 네비게이션 바 상단에 위치하는 대표 이미지/비디오 |
| 2 | **상품 소개 이미지** | 상품을 소개하는 상세 이미지 |

### 1. 상품 대표 이미지/비디오 설정
- `콘텐츠 추가` 버튼 클릭
- 콘텐츠 타입: 이미지 / 비디오
- 이미지: JPG / PNG / WebP 형식, 파일당 4MB 미만
- 동영상: MP4 형식, 파일당 30MB 이하
- 노출 설정: `상시 공개`
- URL 링크(선택): 이미지 클릭 시 설정한 링크로 이동 (비디오는 적용 안 됨)
- 이미지 등록 후 `콘텐츠 추가`로 특정 날짜 기준 이미지/비디오 노출 설정 가능

### 2. 상세 소개 이미지 설정
- `이미지 등록` 버튼 클릭
- 이미지: JPG / PNG / WebP 형식, 파일당 4MB 미만
- URL링크 등록 가능(옵션)
- `추가` 버튼으로 특정 날짜 기준 이미지 노출 설정 가능

---

## STEP 3. 상품 옵션 설정

상품페이지 생성 완료 후, 생성한 상품페이지를 다시 클릭하면 상단에 **[상품옵션]** 탭이 나타납니다.
**[상품등록]** 버튼을 눌러 고객이 실제로 구매할 상품 옵션들을 추가해 주세요.

### 1. 상품 옵션 기본 설정

| 순번 | 항목 | 설명 |
|------|------|------|
| 1 | **상품명(상품옵션명)** | 고객 결제창 및 멤버십 관리에서 노출될 이름 (예: 투자 입문반, 실전 투자반) |
| 2 | **공개 여부** | 해당 옵션을 고객에게 노출할지 결정 |
| 3 | **결제 주기** | 정기 결제 간격 (1, 3, 6, 12개월 등) |
| 4 | **금액(정상가)** | 할인 적용 전 기준 금액. 구독형의 경우 설정한 주기마다 이 금액이 결제 |
| 5 | **포함할 시리즈 선택** | 구매 유저에게 제공할 시리즈 연결. 💡 파트너센터에서 미리 생성한 시리즈만 노출 |
| 6 | **상품 구성** | 유저가 구매 시 받게 되는 시리즈명/혜택 텍스트 |
| 7 | **추천상품 표시** | 체크 시 'BEST PICK' 뱃지 표시 |
| 8 | **태그 추가** | 옵션명 옆 강조 문구 (7자 이내) |

### 2. 프로모션(할인) 설정

- **사용 여부**: [사용] 선택 시 상세 설정 활성화
- **프로모션 타입**:
  - **상시 할인**: 판매 기간 내내 할인가로 판매
  - **구독 회차 한정 할인**: 구독상품 전용. 설정한 회차까지 할인 후 정가 자동 전환
- **할인 적용 방식**:
  - **할인율(%)**: 비율만큼 차감
  - **할인금액(원)**: 금액만큼 차감 (예: 10,000원 - 1,000원 = 9,000원)
- **회차 선택** (구독 회차 한정 시): 1~9회차 중 선택
- **프로모션 기간**: 시작일/종료일 설정. 기간 지나면 자동 비활성

### ⚠️ 상품 변경 시 혜택 적용 로직

- **다운그레이드** (높은 → 낮은 가격): 할인 혜택 즉시 사라짐, 변경된 상품의 정상가로 결제
- **업그레이드** (낮은 → 높은 가격): 변경된 상품의 정상가 기준으로 재산정
- **공통**: 모든 프로모션은 최초 구매 시에만 적용. 상품 변경 후 복구 불가

## 용어 정리

### 메인 상품페이지
어스플러스 각 경로에서 연결되는 상품페이지:
- **구독 탭**: 구독 상품 목록에서 클릭 시 이동
- **오피셜클럽 홈**: 마스터 프로필 페이지에서 연결
- **멤버십 콘텐츠**: 콘텐츠 내에서 상품 페이지 연결

### '모집이 종료된 멤버십입니다'
신청 기간이 종료되면 표시되는 안내 문구. 결제가 제한됨.
