# 단건상품 생성/관리 방법

출처: Slack Canvas `F0AMMPCENU8` (2026-03-16)

---

## STEP 1. 상품 페이지 생성하기

우측 상단 `페이지 생성` 버튼을 눌러서 새로운 상품 페이지를 생성합니다.

---

## STEP 2. 상품 페이지 정보 설정

상품의 노출 조건과 판매 방식을 설정하는 영역입니다.

| 순번 | 항목 | 설명 |
|------|------|------|
| 1 | **히든 처리 여부** | 메인 상품페이지(어스플러스 홈/구독 탭) 외에 특정 프로모션이나 개별 링크로만 접근하게 할 경우 '히든'으로 설정 |
| 2 | **마스터 선택 (오피셜클럽 선택)** | 상품을 판매할 오피셜클럽을 선택 |
| 3 | **페이지명** | 관리자센터에서 노출되는 명칭 (고객에게는 노출되지 않음) |
| 4 | **페이지 공개 여부** | **공개**: 공개 기간에 맞춰 노출. **비공개**: 숨겨짐 (최종 검수 전 비공개 권장) |
| 5 | **공개 기간** | 상품 페이지 접근 가능 기간. 오늘 날짜가 기간 내 포함되어야 활성화 |
| 6 | **상품 타입** | 단건상품: 정해진 기간 동안 멤버십 이용을 위해 한 번만 결제 |
| 7 | **신청 기간** | 구매하기 버튼 동작 기간. 종료 시 '모집이 종료된 멤버십입니다' 표시 |
| 8 | **상품 변경 가능 여부** | 단건 상품은 변경 불가 |
| 9 | **편지글 공개 여부** | '공개' 시 [편지글] 탭 활성화 |

---

## STEP 3. 상품 페이지 이미지 설정

상품 페이지에 이미지/비디오 자료를 등록합니다. 생성 이후에도 수정 가능.

### 사용자 화면 구성

| 순번 | 항목 | 설명 |
|------|------|------|
| 1 | **상품 대표 이미지/비디오** | 네비게이션 바 상단에 위치하는 대표 이미지/비디오 |
| 2 | **상품 소개 이미지** | 상품을 소개하는 상세 이미지 |

### 1. 상품 대표 이미지/비디오 설정
- `콘텐츠 추가` 버튼 클릭
- 콘텐츠 타입: 이미지 / 비디오 (1가지만 설정 가능)
- 이미지: JPG / PNG / WebP 형식, 파일당 4MB 미만
- 동영상: MP4 형식, 파일당 30MB 이하
- 노출 설정: `상시 공개`
- URL 링크(선택): 이미지 클릭 시 링크 이동 (비디오 미적용)
- 특정 날짜 기준 이미지/비디오 노출 설정 가능

### 2. 상세 소개 이미지 설정
- `이미지 등록` 버튼 클릭
- 이미지: JPG / PNG / WebP 형식, 파일당 4MB 미만
- URL링크 등록 가능(옵션)
- 특정 날짜 기준 이미지 노출 설정 가능

---

## STEP 3. 상품 옵션 설정

상품페이지 생성 완료 후, 생성한 상품페이지를 다시 클릭하면 상단에 **[상품옵션]** 탭이 나타납니다.
**[상품등록]** 버튼을 눌러 고객이 실제로 구매할 상품 옵션들을 추가해 주세요.

### 1. 상품 옵션 기본 설정

| 순번 | 항목 | 설명 |
|------|------|------|
| 1 | **상품명(상품옵션명)** | 고객 결제창 및 멤버십 관리에서 노출될 이름 |
| 2 | **공개 여부** | 해당 옵션을 고객에게 노출할지 결정 |
| 3 | **이용 기간 유형** | **일수**: 구매 시점부터 n일 (예: 60일). **기간 지정**: 날짜 선택 (예: 26.03.20~26.05.30) |
| 4 | **금액(정상가)** | 할인 적용 전 기준 금액 |
| 5 | **포함할 시리즈 선택** | 구매 유저에게 제공할 시리즈 연결. 💡 파트너센터에서 미리 생성한 시리즈만 노출 |
| 6 | **상품 구성** | 유저가 구매 시 받게 되는 시리즈명/교재 등 |
| 7 | **추천상품 표시** | 체크 시 'BEST PICK' 뱃지 표시 |
| 8 | **태그 추가** | 옵션명 옆 강조 문구 (7자 이내) |

### 2. 프로모션(할인) 설정

- **사용 여부**: [사용] 선택 시 상세 설정 활성화
- **프로모션 타입**: **상시 할인만 적용** (단건은 구독 회차 한정 불가)
- **할인 적용 방식**:
  - **할인율(%)**: 비율만큼 차감
  - **할인금액(원)**: 금액만큼 차감

### ⚠️ 상품 변경 시 혜택 적용 로직

- **다운그레이드**: 할인 혜택 즉시 사라짐, 정상가로 결제
- **업그레이드**: 변경된 상품의 정상가 기준 재산정
- **공통**: 모든 프로모션은 최초 구매 시에만 적용. 변경 후 복구 불가

## 구독 vs 단건 차이 요약

| 항목 | 구독 | 단건 |
|------|------|------|
| 결제 주기 | 1/3/6/12개월 | 없음 (1회) |
| 이용 기간 | 자동 갱신 | 일수 or 기간 지정 |
| 상품 변경 | 가능/불가 선택 | 불가 |
| 프로모션 | 상시 + 회차한정(1~9회) | 상시만 |
