┌─────────────────────────────────────────┐

│                                          │
│ 📄 상품 페이지: 1개                      │

│ │                                        │
│ │   📦 상품 옵션: 1개                    │
│ │   └── 월간 투자 리포트                 │

- [ ] 마스터 + 상품 페이지 + 옵션 전체 요약
