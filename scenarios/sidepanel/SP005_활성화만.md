# SP005. 활성화만 (사이드패널)

## 메타데이터
- **카테고리**: happy_path
- **난이도**: easy
- **필요 권한**: PRODUCT
- **모드**: 사이드패널 (네비게이터)

## 사용자 입력
> "조조형우 상품 활성화해줘"

## 사이드패널 플로우

### Step 1: 현재 상태 확인 (API 직접)
```
사이드패널: search_masters → get_product_page_list → get_product_list_by_page
→ "조조형우 상품 현재 상태:"
→ "  상품 페이지: 비공개 (INACTIVE)"
→ "  상품 옵션 1개: 월간 투자 리포트 29,900원 (비공개)"
→ "  메인 상품 페이지: INACTIVE"
```

### Step 2: 활성화 (API 직접)
```
사이드패널:
  "아래 항목을 활성화합니다:"
  "  1. 상품 옵션 노출 ON"
  "  2. 상품 페이지 공개"
  "  3. 메인 상품 페이지 ACTIVE"

  ⚠️ 이미지 플레이스홀더면: "이미지가 등록되지 않았습니다."
    [🖼️ 이미지 먼저 등록하기] → ?tab=settings
    [🚀 그래도 활성화하기]

  [🚀 활성화하기]

유저: [🚀 활성화하기]
사이드패널:
  → update_product_display
  → update_product_sequence
  → update_product_page_status (ACTIVE)
  → update_main_product_setting (ACTIVE)
→ "✅ 활성화 완료!"
→ [🌐 고객 화면 확인] → /products/group/{code}
```
