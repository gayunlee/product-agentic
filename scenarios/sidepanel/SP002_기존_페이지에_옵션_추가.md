# SP002. 기존 페이지에 상품 옵션 추가 (사이드패널)

## 메타데이터
- **카테고리**: happy_path
- **난이도**: easy
- **필요 권한**: PRODUCT
- **모드**: 사이드패널 (네비게이터)

## 사용자 입력
> "조조형우 상품에 옵션 하나 더 추가하고 싶어"

## 사이드패널 플로우

### Step 1: 마스터 + 상품 페이지 확인 (API 직접)
```
사이드패널: search_masters("조조형우") → get_product_page_list("35")
→ "✅ 조조형우 (PUBLIC)"
→ "✅ 기존 상품 페이지: 월간 투자 리포트 (코드: 148, ACTIVE)"
→ "  현재 옵션: 1개 (월간 투자 리포트 29,900원)"
```

### Step 2: 바로 옵션 등록 (유저가 페이지에서 직접)
```
사이드패널:
  "상품 옵션을 추가로 등록해주세요."
  [📦 상품 옵션 등록하러 가기]
    → /product/create?productPageId={id}&productType=SUBSCRIPTION&masterId=35
  [✅ 상품 옵션 등록 완료]

유저: 옵션 등록 후 [✅ 완료]
사이드패널: get_product_list_by_page("{id}")
→ "✅ 상품 옵션 2개 확인"
→ "  1. 월간 투자 리포트 29,900원/월"
→ "  2. {새 옵션명} {금액}원/{주기}"
```

### Step 3: 활성화 확인 (API 직접)
```
사이드패널:
  "새 옵션이 추가되었습니다. 공개 상태를 확인합니다..."
  → get_product_list_by_page → 새 옵션의 isDisplay 확인
  → 비공개면: "새 옵션이 비공개 상태입니다. 공개할까요?"
    [🔓 새 옵션 공개하기]  → update_product_display
  → 공개면: "✅ 이미 공개 상태입니다."

  [🌐 고객 화면에서 확인]
```

## 절대 하면 안 되는 것
- 상품 옵션을 API로 직접 등록
- 기존 옵션을 수정/삭제

## 평가 기준
- [ ] 기존 페이지 확인 → 바로 옵션 등록 단계로 (페이지 생성 건너뜀)
- [ ] 새 옵션 등록은 페이지 이동
- [ ] 등록 후 공개 상태 확인 → 필요시 API로 공개 전환
