# SP003. 진단: 상품이 안 보여 (사이드패널)

## 메타데이터
- **카테고리**: happy_path
- **난이도**: medium
- **필요 권한**: PRODUCT + MASTER
- **모드**: 사이드패널 (네비게이터)

## 사용자 입력
> "조조형우 상품이 고객한테 안 보이는데 왜 그래?"

## 사이드패널 플로우

### 전체 자동 (API 직접 — 조회만)
```
사이드패널: "진단을 시작합니다..."

1. search_masters("조조형우")
   → ✅ PUBLIC (공개됨)

2. get_master_detail("35") 또는 관련 API
   → productGroupViewStatus 확인
   → ✅ ACTIVE (메인 상품 페이지 활성화)
   또는 ❌ INACTIVE → "메인 상품 페이지가 비활성화 상태입니다."
     [⚡ ACTIVE로 전환하기]  → update_main_product_setting (API 직접)
     또는 [📋 메인 상품 페이지 관리로 이동] → /product/page/list

3. get_product_page_list("35")
   → 상품 페이지 status 확인
   → ✅ ACTIVE
   또는 ❌ INACTIVE → "상품 페이지가 비공개 상태입니다."
     [⚡ 공개로 전환하기]  → update_product_page_status (API 직접)
     또는 [📄 상품 페이지 상세로 이동] → /product/page/{id}?tab=settings

4. 공개 기간 확인 (startAt/endAt)
   → ✅ 기간 내
   또는 ❌ 기간 외 → "공개 기간이 지났습니다. (2026-01-01 ~ 2026-02-28)"
     [📄 기간 수정하러 가기] → /product/page/{id}?tab=settings

5. get_product_list_by_page("{pageId}")
   → 상품 옵션 isDisplay 확인
   → ✅ 노출 중인 옵션 있음
   또는 ❌ 전부 비공개 → "노출 중인 상품 옵션이 없습니다."
     [⚡ 옵션 공개하기]  → update_product_display (API 직접)
```

### 결과 표시
```
┌─────────────────────────────────────────┐

│                                          │
│ ✅ 마스터 공개 상태: PUBLIC               │
│ ✅ 메인 상품 페이지: ACTIVE              │
│ ❌ 상품 페이지: INACTIVE  ← 원인!        │
│ — 공개 기간: 상시                         │
│ ✅ 상품 옵션: 노출 중                     │
│                                          │
│ 🎯 원인: 상품 페이지가 비공개 상태       │
│                                          │
│ [⚡ 공개로 전환하기]                      │
│ [📄 상품 페이지 상세로 이동]             │
└─────────────────────────────────────────┘
```

## 핵심: 진단은 전부 API 직접
- 조회 API만 사용 → 생성/수정 Tool 호출 금지 (진단 단계에서)
- 원인 발견 시: 상태 변경은 [⚡ 전환] 버튼 (API 직접), 폼 수정은 [📄 이동] 버튼
- 체크리스트 순서: 마스터 → 메인 상품 → 상품 페이지 → 기간 → 옵션
