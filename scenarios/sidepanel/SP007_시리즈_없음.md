# SP007. 시리즈 없을 때 (사이드패널)

## 메타데이터
- **카테고리**: edge_case
- **난이도**: medium
- **필요 권한**: PRODUCT + MASTER
- **모드**: 사이드패널 (네비게이터)

## 사용자 입력
> "조조형우 마스터로 상품 세팅하고 싶어"

## 사이드패널 플로우

### Step 1: 마스터 확인 (API 직접) → ✅ 있음
### Step 2: 시리즈 확인 (API 직접) → ❌ 없음

```
┌─────────────────────────────────────────┐
│ ✅ 조조형우 (PUBLIC, cmsId:35)           │
│ ❌ 시리즈가 없습니다.                    │
│                                          │
│ 상품 옵션에 연결할 시리즈가 필요합니다.  │
│ 파트너센터에서 시리즈를 먼저 생성해주세요.│
│                                          │
│ 💡 시리즈 생성 방법:                     │
│ 1. 좌측 앱 전환 아이콘에서 "파트너" 클릭 │
│ 2. [시리즈 만들기] 버튼 클릭             │
│ 3. 카테고리, 제목, 설명, 썸네일 입력     │
│                                          │
│ [📝 파트너센터로 이동]                   │
│                                          │
│ 시리즈 생성 완료되면 알려주세요.         │
│ [✅ 시리즈 생성 완료]                    │
└─────────────────────────────────────────┘
```

### [완료] 후
```
사이드패널: get_series_list("35")
→ 시리즈 있으면: "✅ 시리즈 {N}개 확인" → Step 3(상품 페이지 생성)으로
→ 아직 없으면: "아직 시리즈가 없습니다. 파트너센터에서 생성해주세요."
```

## 핵심
- 시리즈 없이 상품 옵션 등록 불가 → 여기서 반드시 중단
- 파트너센터는 외부 앱 → 직접 URL 링크가 아닌 앱 전환 안내
- 시리즈 생성 방법을 간략히 가이드

## 평가 기준
- [ ] 시리즈 없으면 상품 페이지 생성으로 넘어가지 않음
- [ ] 파트너센터 이동 안내 (앱 전환 아이콘)
- [ ] [완료] 후 시리즈 재확인
- [ ] 확인 후 자동으로 다음 단계 진행
