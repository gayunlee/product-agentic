# SP004. 마스터 없을 때 (사이드패널)

## 메타데이터
- **카테고리**: edge_case
- **난이도**: medium
- **필요 권한**: PRODUCT + MASTER
- **모드**: 사이드패널 (네비게이터)

## 사용자 입력
> "박가윤 마스터로 상품 세팅하고 싶어"

## 사이드패널 플로우

### Step 1: 마스터 확인 (API 직접)
```
사이드패널: search_masters("박가윤") + get_master_groups("박가윤")
→ 둘 다 결과 없음

┌─────────────────────────────────────────┐
│ ❌ "박가윤" 마스터가 등록되어 있지       │
│    않습니다.                             │
│                                          │
│ 상품 세팅을 위해 아래 순서로 진행해야    │
│ 합니다:                                  │
│                                          │
│ 1️⃣ 마스터 그룹 생성                     │
│ 2️⃣ 오피셜클럽 생성                      │
│ 3️⃣ 시리즈 생성 (파트너센터)             │
│ 4️⃣ 상품 페이지 + 옵션 세팅             │
│                                          │
│ 먼저 마스터 그룹을 생성할까요?           │
│                                          │
│ [🏗️ 마스터 그룹 생성하기]               │
│  (에이전트가 이름만으로 바로 생성)       │
│                                          │
│ 또는 오피셜클럽부터 직접 만드시려면:     │
│ [🏠 오피셜클럽 생성하러 가기]            │
│  → /official-club/create                 │
└─────────────────────────────────────────┘
```

### Step 1-1: 마스터 그룹 생성 (API 직접 — 이름만)
```
유저: [🏗️ 마스터 그룹 생성하기] 클릭
사이드패널: create_master_group(name="박가윤")
→ "✅ 마스터 그룹 '박가윤' 생성 완료"

→ "다음으로 오피셜클럽을 생성해주세요."
→ "마스터 선택에서 '박가윤'을 선택해주세요."

  [🏠 오피셜클럽 생성하러 가기] → /official-club/create
  [✅ 오피셜클럽 생성 완료]
```

### Step 1-2: 오피셜클럽 생성 (유저가 페이지에서 직접)
```
유저: 오피셜클럽 생성 후 [✅ 완료]
사이드패널: search_masters("박가윤")
→ "✅ 오피셜클럽: 박가윤 ({publicType}, cmsId:{id})"
→ Step 2 (시리즈 확인)로 자동 진행
```

## 핵심
- 마스터 그룹 생성은 이름만 → API 직접 가능
- 오피셜클럽 생성은 필드 15개+ 이미지 → 페이지 이동
- 전체 로드맵(1→2→3→4) 미리 보여줘서 유저가 얼마나 남았는지 알게 함

## 평가 기준
- [ ] 마스터 없음을 명확히 안내
- [ ] 전체 로드맵 표시 (4단계)
- [ ] 마스터 그룹 생성은 API 직접 (이름만)
- [ ] 오피셜클럽 생성은 페이지 이동
- [ ] [완료] 후 API로 재확인
