# #15 상품 페이지 히든 처리 기능이 뭐야?

## 분류
- 유저 의도: 도메인 질문 (explain_hidden)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음

---

## Case A: 정상

```
유저: 상품 페이지 히든 처리 기능이 뭐야?
에이전트: 히든 처리를 하면 상품 페이지가 구독 탭이나 오피셜클럽 홈에 노출되지 않고,
         URL(/products/group/{code})로 직접 접근할 때만 볼 수 있습니다.

         특정 프로모션이나 개별 링크로만 접근하게 하고 싶을 때 사용합니다.

         참고:
         - 히든 페이지는 기간 충돌 검사에서 제외됩니다
         - 히든 페이지의 status는 ACTIVE 유지 → URL 접근 가능
         - 히든(목록 숨김, URL 접근 가능)과 비공개(INACTIVE, URL 접근도 불가)는 다른 개념입니다
```

---

## 검증 포인트 (eval 기준)

1. URL 직접 접근만 가능하다는 점 언급
2. 구독 탭/오피셜클럽 홈에 미노출 언급
3. 기간 충돌 검사 제외 언급
4. 노출 체인 일부 면제 언급
