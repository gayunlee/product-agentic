## 분류
- 유저 의도: 도메인 질문 (explain_master_vs_club)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트

## Case A: 정상

```
유저: 이거 구독 페이지에서 노출되는 건 마스터야 오피셜클럽이야?
에이전트: 오피셜클럽 단위로 노출됩니다.

         관리자센터에서는 "마스터"라고 표시되는 경우가 있지만,
         어스플러스에서 고객에게 보이는 건 전부 오피셜클럽 단위입니다.

         구독 탭, 오피셜클럽 홈, 멤버십 콘텐츠 모두 오피셜클럽 기준입니다.
```
