# #1 상품페이지 새로 만들려고

## 분류
- 유저 의도: 수행 (guide_create_page)
- 에이전트: 오케스트레이터 → 수행 에이전트 + 검증 에이전트

## 사전조건 체인
```
마스터 그룹 → 오피셜클럽(cmsId) → 시리즈 → 상품 페이지
```

## 에이전트 역할
- 오케스트레이터: "만들려고" → 수행 요청으로 분류 → 수행 에이전트에 전달
- 검증 에이전트: check_prerequisites("create_product_page", context) → missing 반환
- 수행 에이전트: missing 기반으로 유저에게 질문 → 정보 수집 → 페이지 이동 안내

---

## Case A: 정상 플로우 (오피셜클럽 1개, 시리즈 있음)

```
유저: 상품페이지 새로 만들려고
에이전트: 어떤 오피셜클럽에 만드시겠어요?
유저: 조조형우
에이전트: 조조형우 오피셜클럽 확인했습니다. (cmsId: 35)
         시리즈 2개 확인했습니다: "투자 기초 시리즈", "경제 분석 리포트"
         구독상품과 단건상품 중 어떤 타입으로 만드시겠어요?
유저: 구독
에이전트: 상품 페이지 생성 화면으로 이동해주세요.
         마스터에서 "조조형우"를 선택하고, 정보 설정과 이미지까지 등록해주세요.
         [페이지 생성으로 이동 →] (/product/page/create)
         완료하시면 알려주세요!
유저: 다 했어
에이전트: (get_product_page_list 조회) 새 상품 페이지 "월간 투자 리포트" 확인했습니다.
         상품 옵션도 등록하시겠어요?
```

## Case B: 오피셜클럽 여러 개

```
유저: 상품페이지 새로 만들려고
에이전트: 어떤 오피셜클럽에 만드시겠어요?
유저: 조조형우
에이전트: "조조형우" 오피셜클럽이 2개 있습니다:
         1. 조조형우의 투자클럽 (cmsId: 35)
         2. 조조형우의 경제교실 (cmsId: 88)
         어떤 오피셜클럽에 만드시겠어요?
유저: 1번
에이전트: 조조형우의 투자클럽 선택했습니다. 시리즈 확인 중...
         (이하 Case A와 동일)
```

## Case C: 오피셜클럽 없음 (마스터 그룹은 있음)

```
유저: 상품페이지 새로 만들려고
에이전트: 어떤 오피셜클럽에 만드시겠어요?
유저: 에이전트 테스트 신규
에이전트: "에이전트 테스트 신규" 오피셜클럽이 없습니다.
         상품 페이지를 만들려면 오피셜클럽이 먼저 필요합니다.
         오피셜클럽을 생성하시겠어요?
         [오피셜클럽 생성으로 이동 →] (/official-club/create)
유저: 근데 마스터 그룹도 없는데?
에이전트: (get_master_groups 조회) 맞습니다, 마스터 그룹도 없네요.
         마스터 그룹 → 오피셜클럽 → 시리즈 순서로 만들어야 합니다.
         먼저 마스터 그룹을 생성하시겠어요?
         [마스터 관리로 이동 →] (/master)
```

## Case D: 시리즈 없음

```
유저: 상품페이지 새로 만들려고
에이전트: 어떤 오피셜클럽에 만드시겠어요?
유저: 봄날의 햇살
에이전트: 봄날의 햇살 오피셜클럽 확인했습니다. (cmsId: 60)
         시리즈를 확인하는 중...
         시리즈가 없습니다. 상품 옵션에 시리즈 연결이 필수이므로,
         파트너센터에서 시리즈를 먼저 생성해주세요.
         [파트너센터로 이동 →] (master.us-insight.com)
         시리즈 생성 후 다시 알려주세요!
```

## Case E: 컨텍스트에 이미 마스터 정보가 있는 경우

```
(사이드패널 URL: /product?masterId=35)
유저: 상품페이지 새로 만들려고
에이전트: 현재 조조형우(cmsId: 35) 페이지에 계시네요.
         이 오피셜클럽에 만드시겠어요?
유저: ㅇㅇ
에이전트: 시리즈 2개 확인했습니다. 구독/단건 중 어떤 타입으로 만드시겠어요?
```

## Case F: 완전 신규 (마스터 그룹부터)

```
유저: 상품페이지 새로 만들려고
에이전트: 어떤 오피셜클럽에 만드시겠어요?
유저: 김영익
에이전트: "김영익" 오피셜클럽이 없습니다.
         마스터 그룹을 확인하는 중...
         "김영익" 마스터 그룹도 없습니다.

         상품 페이지를 만들려면 아래 순서로 준비가 필요합니다:
         1. 마스터 그룹 생성 → [마스터 관리로 이동 →] (/master)
         2. 오피셜클럽 생성 → [오피셜클럽 생성으로 이동 →] (/official-club/create)
         3. 시리즈 생성 → [파트너센터로 이동 →] (master.us-insight.com)

         마스터 그룹부터 만드시겠어요?
유저: ㅇㅇ
에이전트: 마스터 그룹 이름을 알려주세요.
유저: 김영익
에이전트: (create_master_group("김영익")) 마스터 그룹 "김영익" 생성 완료했습니다.
         다음으로 오피셜클럽을 생성해주세요.
         [오피셜클럽 생성으로 이동 →] (/official-club/create)
         마스터 선택에서 "김영익"을 선택하고 프로필을 등록해주세요.
         완료하시면 알려주세요!
유저: 다 했어
에이전트: (search_masters("김영익")) 오피셜클럽 "김영익" 확인했습니다. (cmsId: 200)
         다음으로 시리즈를 생성해주세요.
         [파트너센터로 이동 →] (master.us-insight.com)
         시리즈 생성 후 알려주세요!
유저: 시리즈 만들었어
에이전트: (get_series_list("200")) 시리즈 1개 확인했습니다: "김영익의 경제 전망"
         이제 상품 페이지를 만들 수 있습니다.
         구독상품과 단건상품 중 어떤 타입으로 만드시겠어요?
```

## 검증 포인트 (eval 기준)

1. 오피셜클럽 없이 상품 페이지 생성 시도 → 반드시 차단
2. 시리즈 없이 진행 시도 → 반드시 차단
3. 오피셜클럽 여러 개 → 유저에게 선택 요청 (임의 선택 금지)
4. 컨텍스트에 masterId 있어도 유저 확인 필수
5. 완전 신규 시 마스터 그룹 → 오피셜클럽 → 시리즈 순서 안내
6. 각 단계 완료 후 API로 재확인 (self-verification)
