# #36 상품페이지 공개해줘. ㅇㅇ상품이랑 ㅇㅇ상품들 보여야돼

## 분류
- 유저 의도: 수행 (toggle_page_status + verify_options)
- 에이전트: 오케스트레이터 → 수행 에이전트 + 검증 에이전트
- 핵심: 공개 전에 유저가 언급한 상품 옵션들의 isDisplay 확인

---

## Case A: 일부 상품 비공개

```
유저: 상품페이지 공개해줘. 월간구독이랑 연간구독 보여야돼
에이전트: 어떤 오피셜클럽의 상품 페이지인가요?
유저: 조조형우
에이전트: (목록 조회 → 선택)
         공개 전에 확인합니다...
         (상품 옵션 조회)
         - 월간 구독 — 공개 ✅
         - 연간 구독 — 비공개 ❌

         "연간 구독"이 비공개 상태입니다.
         페이지를 공개해도 이 상품은 보이지 않습니다.
         연간 구독도 공개로 변경하시겠어요?
유저: ㅇㅇ
에이전트: (연간 구독 공개 처리 + 페이지 공개 처리)
         완료했습니다.
         [상품 페이지 확인하기 →] (/product/page/{id}?tab=options)
```

## Case B: 전부 공개 상태

```
유저: 상품페이지 공개해줘. 월간구독이랑 연간구독 보여야돼
에이전트: (상품 옵션 조회)
         월간 구독, 연간 구독 모두 공개 상태입니다. ✅
         상품 페이지를 공개 처리하시겠어요?
         [공개 처리하기] [직접 수정하기 →] (/product/page/{id}?tab=settings)
유저: (공개 처리하기)
에이전트: 공개 처리 완료했습니다.
         [상품 페이지 확인하기 →] (/product/page/{id}?tab=settings)
```

---

## 검증 포인트 (eval 기준)

1. 공개 전 유저가 언급한 상품 옵션의 isDisplay 확인
2. 비공개 상품이 있으면 경고 + 함께 공개할지 질문
3. 페이지 공개 + 상품 공개 순서 처리
