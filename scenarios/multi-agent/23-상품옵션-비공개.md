# #23 상품 비공개해줘

## 분류
- 유저 의도: 수행 (toggle_product_display)
- 에이전트: 오케스트레이터 → 수행 에이전트
- 방식: API 직접 실행 또는 수정 페이지 이동 (유저 선택)
- 참고: "상품" = 상품 옵션 (상품 페이지가 아님)

## 사전조건 체인
```
오피셜클럽(cmsId) → 상품 페이지 특정 → 상품 옵션 특정
```

---

## Case A: 정상

```
유저: 상품 비공개해줘
에이전트: 어떤 오피셜클럽의 상품인가요?
유저: 조조형우
에이전트: 어떤 상품 페이지인가요?
         1. 월간 투자 리포트 (코드: 148)
         2. 단건 특강 (코드: 155)
유저: 1번
에이전트: (상품 옵션 목록 조회)
         어떤 상품 옵션을 비공개 하시겠어요?
         1. 월간 구독 (29,900원) — 공개 ✅
         2. 연간 구독 (299,000원) — 공개 ✅
유저: 2번
에이전트: 연간 구독을 비공개 처리하시겠어요?
         [비공개 처리하기] [직접 수정하기 →] (/product/page/{id}?tab=options)
유저: (비공개 처리하기)
에이전트: 비공개 처리 완료했습니다.
         [상품 페이지 확인하기 →] (/product/page/{id}?tab=options)
```

---

## 검증 포인트 (eval 기준)

1. "상품" = 상품 옵션으로 이해
2. 오피셜클럽 → 상품 페이지 → 상품 옵션 순서로 특정
3. API 실행 버튼 + 직접 수정 버튼 둘 다 제공
4. 변경 전 유저 확인 필수
5. 완료 후 확인 페이지 이동
