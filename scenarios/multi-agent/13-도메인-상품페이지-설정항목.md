# #13 상품페이지 하나 설정할때 다른 설정 뭐뭐가 필요해?

## 분류
- 유저 의도: 도메인 질문 (explain_prerequisites)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음 (지식 기반 답변)

---

## Case A: 정상

```
유저: 상품페이지 하나 설정할때 다른 설정 뭐뭐가 필요해?
에이전트: 상품 페이지를 만들기 전에 준비해야 할 것:
         1. 마스터 그룹 (없으면 /master에서 생성)
         2. 오피셜클럽 (없으면 /official-club/create에서 생성)
         3. 시리즈 (파트너센터에서 생성)

         상품 페이지 생성 시 설정하는 항목:
         1. 히든 처리 여부
         2. 마스터 선택 (=오피셜클럽)
         3. 페이지명 (관리자용, 고객 미노출)
         4. 공개 여부
         5. 공개 기간
         6. 상품 타입 (구독/단건)
         7. 신청 기간
         8. 상품 변경 가능 여부
         9. 편지글 공개 여부

         페이지 생성 후에는 이미지 등록 + 상품 옵션 등록이 필요합니다.
```

---

## 검증 포인트 (eval 기준)

1. 사전조건 3가지 (마스터 그룹, 오피셜클럽, 시리즈) 언급
2. 9개 설정 항목 안내
3. 생성 후 추가 작업 (이미지, 상품 옵션) 언급
