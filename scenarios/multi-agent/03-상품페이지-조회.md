# #3 ㅇㅇ 오피셜클럽 상품페이지 중에 'ㅁㅁ' 있지?

## 분류
- 유저 의도: 수행 (query_page)
- 에이전트: 오케스트레이터 → 수행 에이전트

## 사전조건 체인
```
오피셜클럽(cmsId) → 상품 페이지 목록 조회
```

---

## Case A: 정상 (있음)

```
유저: 조조형우 상품페이지 중에 '월간 투자' 어쩌고 하는 상품페이지 있지?
에이전트: (search_masters → get_product_page_list)
         네, 있습니다.
         - 월간 투자 리포트 (코드: 148, 구독, 공개)
         상세 정보나 수정이 필요하시면 알려주세요.
```

## Case B: 없음

```
유저: 봄날의 햇살 상품페이지 중에 '프리미엄' 있지?
에이전트: (search_masters → get_product_page_list)
         봄날의 햇살 상품 페이지 2개를 확인했는데, '프리미엄'이 포함된 페이지는 없습니다.
         현재 페이지 목록:
         1. 단건 상품 (코드: 141, 히든)
         2. 단건상품 (코드: 124, 공개)
```

## Case C: 오피셜클럽명 없이 질문

```
유저: '월간 투자' 상품페이지 있어?
에이전트: 어떤 오피셜클럽의 상품 페이지를 찾으시는 건가요?
```

---

## 검증 포인트 (eval 기준)

1. 오피셜클럽 + 키워드가 메시지에 포함되면 바로 조회
2. 없으면 전체 목록 보여주기
3. 오피셜클럽 미특정 시 질문
