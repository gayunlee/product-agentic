# #19 상품 변경 못하게 하려면 어떻게 해야돼?

## 분류
- 유저 의도: 도메인 질문 (explain_changeable)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음

---

## Case A: 정상

```
유저: 상품 변경 못하게 하려면 어떻게 해야돼?
에이전트: 상품 페이지 설정에서 "상품 변경 가능 여부"를 비활성화하시면 됩니다.
         고객이 마이페이지에서 상품 옵션을 변경할 수 없게 됩니다.

         참고: 단건상품은 기본적으로 변경 불가입니다. 구독상품만 해당됩니다.
```

---

## 검증 포인트 (eval 기준)

1. isChangeable 설정 안내
2. 단건 = 기본 변경 불가 언급
