# #11 오피셜클럽명 필드에 뭐가 많은데 다 뭐야?

## 분류
- 유저 의도: 도메인 질문 (explain_fields)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음 (지식 기반 답변)

---

## Case A: 정상

```
유저: 이거 오피셜클럽명 필드에 뭐가 많은데 이게 다 뭐야? 고객화면 표시용 관리자센터 표시용 이런게 다 뭐야?
에이전트: 이름 관련 필드가 3개 있고, 각각 용도가 다릅니다:

         1. **오피셜클럽명 - 고객 표시명** (name, 최대 17자)
            → 어스플러스에서 고객이 보는 모든 곳에 표시됩니다. (카드, 목록, 콘텐츠 작성자명 등)

         2. **오피셜클럽명 - 관리자 표시명** (displayName)
            → 관리자센터, 파트너센터 내부에서만 보입니다. 고객에게는 안 보여요.

         3. **오피셜클럽 타이틀** (clubName, 최대 100자)
            → 오피셜클럽 페이지 상단 타이틀에 표시됩니다.

         name은 짧은 이름(카드, 목록 등), clubName은 긴 타이틀(클럽 페이지 상단) 용도입니다.
```

---

## 검증 포인트 (eval 기준)

1. name / displayName / clubName 3개 구분 정확
2. 각각 어디에 노출되는지 안내
3. name=고객용, displayName=내부용, clubName=클럽 타이틀 구분
