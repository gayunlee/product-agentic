# #7 왜 마스터가 안보여?

## 분류
- 유저 의도: 수행 (diagnose_visibility)
- 에이전트: 오케스트레이터 → 수행 에이전트 + 검증 에이전트
- 본질: #6과 동일한 노출 체인 진단. 유저 표현만 "상품페이지" 대신 "마스터"

## 유저가 말하는 "마스터가 안 보여"
= 구독 탭(/subscribe)에서 해당 마스터가 목록에 없다는 의미

## 노출 조건
1. 마스터 PUBLIC (마스터 관리에서 공개)
2. 메인 상품 페이지 ACTIVE (메인 상품 페이지 관리에서 공개)

---

## 용어 변환 규칙
- 어스플러스에서 "마스터" = 오피셜클럽
- 유저가 "마스터가 안 보여"라고 하면 → "오피셜클럽을 말씀하시는 것 같네요. 어떤 오피셜클럽인가요?"

---

## Case A: 마스터 비공개

```
유저: 왜 마스터가 안보여?
에이전트: 오피셜클럽을 말씀하시는 것 같네요. 어떤 오피셜클럽인가요?
유저: 김영익
에이전트: (diagnose_visibility)
         ❌ 마스터 상태: PENDING

         마스터가 아직 준비중(PENDING) 상태입니다.
         PUBLIC으로 전환해야 구독 탭에 노출됩니다.
         [오피셜클럽 관리로 이동 →] (/official-club/{cmsId})
```

## Case B: 마스터 PUBLIC인데 메인 상품 비활성

```
유저: 왜 마스터가 안보여?
에이전트: 어떤 오피셜클럽인가요?
유저: 봄날의 햇살
에이전트: (diagnose_visibility)
         ✅ 마스터 상태: PUBLIC
         ❌ 메인 상품 페이지: INACTIVE ← 원인

         마스터는 공개 상태이지만, 메인 상품 페이지가 비활성화되어 있어
         구독 탭에 노출되지 않습니다.
         메인 상품 페이지를 활성화하시겠어요?
```

---

## 검증 포인트 (eval 기준)

1. "마스터가 안 보여" = 구독 탭 미노출로 이해
2. 마스터 PUBLIC + 메인 상품 ACTIVE 두 조건 모두 체크
3. 첫 번째 실패 지점 안내
