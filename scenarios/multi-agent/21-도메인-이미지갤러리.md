# #21 한 행에 이미지 여러개 입력하는 거 뭐야?

## 분류
- 유저 의도: 도메인 질문 (explain_image_gallery)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음

---

## Case A: 정상

```
유저: 이거 상품페이지에 한 행에 이미지 여러개 입력하는 거 뭐야? 어떻게 뜨는 거야?
에이전트: 같은 위치에 날짜별로 다른 이미지를 표시하는 기능입니다.

         - 기본 이미지 1개 (상시 표시)
         - 추가 이미지 N개 (특정 기간에만 기본 이미지 대신 표시)

         예: 평소에는 기본 이미지, 프로모션 기간에는 프로모션 이미지로 자동 교체

         주의:
         - 추가 이미지끼리 날짜가 겹치면 안 됩니다
         - 기본 이미지를 삭제하려면 추가 이미지를 먼저 모두 삭제해야 합니다
```

---

## 검증 포인트 (eval 기준)

1. 기본 + 추가 이미지 구조 설명
2. 날짜별 자동 교체 언급
3. 날짜 겹침 불가 + 삭제 순서 제약 언급
