# #9 상품페이지에 왜 편지글이 안보여?

## 분류
- 유저 의도: 수행 (diagnose_letter)
- 에이전트: 오케스트레이터 → 수행 에이전트

## 편지글 노출 — 두 가지 독립 요소

### 편지글 탭 (섹션 목록)
- 조건: `displayLetterStatus === 'ACTIVE'` + 편지 3개 이상
- 설정 위치: 상품 페이지 설정 (/product/page/{id})

### 편지 슬라이더 (고정 편지 캐러셀)
- 조건: `viewType === 'PRODUCT_GROUP'`인 편지가 1개 이상
- 설정 위치: 편지글 관리 (/cs/letter)에서 뷰타입 지정
- 없으면 자동 숨김 (별도 플래그 없음)

**두 가지가 독립적 → 슬라이더만 보이고 탭은 안 보이는 경우도 가능**

---

## Case A: displayLetterStatus 비활성

```
유저: 상품페이지에 왜 편지글이 안보여?
에이전트: 어떤 오피셜클럽의 상품 페이지인가요?
유저: 봄날의 햇살
에이전트: (상품 페이지 상세 조회)
         편지글 공개 설정이 비활성(INACTIVE) 상태입니다.
         편지글 탭을 표시하려면 공개로 변경해야 합니다.
         [상품 페이지 수정으로 이동 →] (/product/page/{id}?tab=settings)
```

## Case B: 활성인데 편지 3개 미만

```
유저: 편지글 탭이 왜 안 보여?
에이전트: 어떤 오피셜클럽의 상품 페이지인가요?
유저: 조조형우
에이전트: (상품 페이지 조회 + 편지 목록 조회)
         편지글 공개 설정은 활성화되어 있지만,
         편지가 2개뿐입니다. 3개 이상이어야 탭이 표시됩니다.
```

## Case C: 편지글 슬라이더가 안 보임 (#29와 연결)

```
유저: 편지글 탭은 보이는데 편지글 슬라이더는 왜 안보여?
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (GET /v3/letters/by-master/{masterId}?viewType=PRODUCT_GROUP)
         뷰타입이 'PRODUCT_GROUP'으로 지정된 편지가 없습니다.
         슬라이더에 표시하려면 편지글 관리에서 뷰타입을 변경해주세요.
         [편지글 관리로 이동 →] (/cs/letter)
```

---

## 검증 포인트 (eval 기준)

1. "편지글 안 보여" → 탭인지 슬라이더인지 구분해서 진단
2. 탭: displayLetterStatus + 편지 3개 이상
3. 슬라이더: viewType === 'PRODUCT_GROUP' 편지 유무
4. 두 가지가 독립적임을 인지
