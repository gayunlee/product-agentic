# #28 왜 오피셜클럽에서 응원하기 메뉴가 안보여?

## 분류
- 유저 의도: 수행 (diagnose_donation)
- 에이전트: 오케스트레이터 → 수행 에이전트

## 진단 기준
- API: `GET /v1/master/{cmsId}/community/category`
- 핵심 필드: `donationActiveStatus`
- ACTIVE → 응원하기 메뉴 표시
- INACTIVE/PENDING → 안 보임

---

## Case A: 비활성

```
유저: 왜 오피셜클럽에서 응원하기 메뉴가 안보여?
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (API 조회)
         응원하기가 비활성화 상태입니다.
         활성화하시겠어요?
         [응원하기 관리로 이동 →] (/donation)
```

## Case B: 이미 활성인데 안 보인다고 할 때

```
유저: 응원하기가 안 보여
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (API 조회)
         응원하기는 활성화(ACTIVE) 상태입니다.
         정상적으로 노출되어야 합니다.
         모바일에서는 별도 탭 없이 카테고리 페이지에서만 접근 가능합니다.
```

---

## 검증 포인트 (eval 기준)

1. donationActiveStatus 체크
2. PC/모바일 접근 방식 차이 안내
