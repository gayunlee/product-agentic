# #20 상품 삭제하고 싶은데 왜 삭제 버튼이 안떠?

## 분류
- 유저 의도: 도메인 질문 (explain_delete_rule)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트

---

## Case A: 정상

```
유저: 이거 상품페이지에 있는 상품 삭제하고 싶은데 왜 삭제 버튼이 안떠? 어떤건 뜨는데?
에이전트: 활성 구독이 있거나 거래 내역이 있는 상품은 삭제할 수 없습니다.
         삭제 가능한 상품에만 삭제 버튼이 표시됩니다.

         삭제가 안 되는 경우, 비공개 처리를 추천드립니다.
         비공개로 변경하시겠어요?
```

---

## 검증 포인트 (eval 기준)

1. 내부 필드명(isDeletable) 노출 금지 — 운영매니저 용어로 설명
2. 삭제 불가 사유: 활성 구독, 거래 내역
3. 대안 제시: 비공개 처리
