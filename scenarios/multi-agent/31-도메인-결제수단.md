- 유저 의도: 도메인 질문 (explain_payment_methods)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음
- 포함: #31 결제수단 다른가, #32 가상계좌 가능한 유형, #33 정기결제 가상계좌, #34 카카오페이 정기결제

---

## Case A: 전체 안내

```
유저: 상품 결제 수단이 상품유형에 따라 다른가?
에이전트: 네, 다릅니다.
         - 구독상품: 신용카드, 카카오페이 (2개)
         - 단건상품: 신용카드, 카카오페이, 네이버페이, 삼성페이, 가상계좌 (5개)

         정기결제(구독)는 자동결제가 되어야 해서 카드와 카카오페이만 가능합니다.
         가상계좌는 단건상품만 가능하고, 선물하기/응원하기에는 사용할 수 없습니다.
```

## Case B: 가상계좌 질문

```
유저: 가상계좌가 가능한 상품유형이뭐야?
에이전트: 단건상품만 가상계좌 결제가 가능합니다.
         구독상품, 선물하기, 응원하기에는 사용할 수 없습니다.
```

## Case C: 정기결제 가상계좌

```
유저: 정기결제는 가상계좌 결제가 가능하던가?
에이전트: 불가능합니다.
         정기결제(구독)는 자동결제가 필요해서 신용카드와 카카오페이만 지원합니다.
```

## Case D: 카카오페이 정기결제

```
유저: 카카오페이도 정기결제되나?
에이전트: 네, 가능합니다.
         카카오페이 자동결제로 구독상품 정기결제를 지원합니다.
```

---

## 검증 포인트 (eval 기준)

1. 구독=2개, 단건=5개 정확
2. 가상계좌=단건만
3. 카카오페이=구독 가능(자동결제)
4. 네이버페이/삼성페이=구독 불가
