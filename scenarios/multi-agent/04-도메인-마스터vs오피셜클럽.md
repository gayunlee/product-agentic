## Case A: 정상

```
유저: 마스터는 뭐고 오피셜클럽은 뭐야 뭐가달라?
에이전트: 마스터 그룹과 오피셜클럽은 상위-하위 관계입니다.

         - **마스터 그룹**: 상위 그룹 개념입니다. 이름만 있고, 마스터 관리(/master)에서 생성합니다.
         - **오피셜클럽**: 마스터 그룹 하위의 실제 운영 단위입니다. 프로필, 상품, 게시판, 시리즈 등이 모두 오피셜클럽 기준으로 관리됩니다.

         한 마스터 그룹 아래 여러 오피셜클럽을 만들 수 있습니다.
         상품 페이지 생성 시 "마스터 선택"이라고 표시되지만, 실제로는 오피셜클럽을 선택하는 것입니다.
```

---

## 검증 포인트 (eval 기준)

1. API 호출 없이 답변
2. 마스터 그룹 ≠ 오피셜클럽 구분 정확
3. "상품 페이지의 마스터 선택 = 오피셜클럽 선택" 언급
