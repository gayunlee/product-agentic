# #17 노출은 하는데 구매는 막고 싶은데 어떻게 해?

## 분류
- 유저 의도: 도메인 질문 (explain_apply_period)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음

---

## Case A: 정상

```
유저: 상품 페이지 노출은 하는데 구매는 막고 싶은데 어떻게 해? 신청은 못하게 할 건데
에이전트: 신청 기간을 설정하시면 됩니다.

         신청 기간이 종료되면 상품 페이지는 보이지만
         "모집이 종료된 멤버십입니다"라는 안내와 함께 구매 버튼이 비활성화됩니다.

         상품 페이지 설정에서 "신청 기간"을 과거 날짜로 설정하거나,
         원하는 종료일을 지정하시면 됩니다.
```

---

## 검증 포인트 (eval 기준)

1. 공개 기간 ≠ 신청 기간 구분
2. 신청 기간 종료 → "모집이 종료된 멤버십입니다" 안내 언급
3. 구매 버튼 비활성화 언급
