# #10 편지글 세팅 바꾸고 싶어

## 분류
- 유저 의도: 수행 (guide_edit_letter)
- 에이전트: 오케스트레이터 → 수행 에이전트

## 세팅 종류 (2가지)
1. 상품 페이지의 편지글 탭 공개/비공개 → `/product/page/{id}?tab=settings`
2. 편지글 라운지/슬라이더 노출 관리 → `/cs/letter`

---

## Case A: 어떤 세팅인지 불명확

```
유저: 편지글 세팅 바꾸고 싶어
에이전트: 어떤 세팅을 변경하시겠어요?
         1. 상품 페이지의 편지글 탭 공개/비공개 → 상품 페이지 설정
         2. 편지글 라운지/슬라이더 노출 관리 → 편지글 관리
유저: 1번
에이전트: 어떤 오피셜클럽의 상품 페이지인가요?
유저: 조조형우
에이전트: (상품 페이지 목록 조회 → 선택)
         [상품 페이지 수정으로 이동 →] (/product/page/{id}?tab=settings)
```

## Case B: 라운지 노출 관리

```
유저: 편지글 라운지 세팅 바꾸고 싶어
에이전트: [편지글 관리로 이동 →] (/cs/letter)
         라운지 노출은 편지의 뷰타입을 'LOUNGE'로 변경하면 됩니다. (최대 8개)
```

---

## 검증 포인트 (eval 기준)

1. "편지글 세팅" → 2가지 중 어떤 건지 확인
2. 상품 페이지 설정 vs 편지글 관리 구분
