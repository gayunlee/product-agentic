# #8 오피셜클럽 게시판이 왜 안 보여?

## 분류
- 유저 의도: 수행 (diagnose_board)
- 에이전트: 오케스트레이터 → 수행 에이전트

## 진단 기준
- API: `GET /v1/master/{cmsId}/community/category`
- 핵심 필드: `postBoardActiveStatus`
- ACTIVE + 게시글 있음 → 정상 표시
- ACTIVE + 게시글 없음 → "아직 작성된 게시글이 없어요"
- INACTIVE → "게시판을 준비하고 있어요"

---

## Case A: 게시판 비활성

```
유저: 오피셜클럽 게시판이 왜 안 보여?
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (GET /v1/master/{cmsId}/community/category)
         게시판이 비활성화(INACTIVE) 상태입니다.
         고객에게 "게시판을 준비하고 있어요"로 표시됩니다.
         활성화하시겠어요?
         [게시판 설정으로 이동 →] (/board/setting)
```

## Case B: 게시판 활성인데 게시글 없음

```
유저: 게시판이 안 보여
에이전트: 어떤 오피셜클럽인가요?
유저: 봄날의 햇살
에이전트: (GET /v1/master/{cmsId}/community/category)
         게시판은 활성화(ACTIVE) 상태입니다.
         다만 게시글이 없어서 고객에게 "아직 작성된 게시글이 없어요"로 표시됩니다.
         게시글이 작성되면 정상 표시됩니다.
```

## Case C: 정상인데 안 보인다고 할 때

```
유저: 게시판이 안 보이는데?
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (GET /v1/master/{cmsId}/community/category)
         게시판은 활성화(ACTIVE) 상태이고, 게시글도 있습니다.
         정상적으로 노출되어야 합니다.
         혹시 모바일에서 확인하신 건가요? 모바일에서는 게시판 탭이 isActiveBoard일 때만 표시됩니다.
```

---

## 검증 포인트 (eval 기준)

1. postBoardActiveStatus 체크
2. 활성인데 게시글 없는 경우도 안내
3. PC/모바일 탭 표시 차이 인지
