# #14 상시로 설정하고 기간 설정한 페이지가 각각 있으면 뭐가 먼저 노출돼?

## 분류
- 유저 의도: 도메인 질문 (explain_visibility_priority)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음 (지식 기반 답변)

---

## Case A: 정상

```
유저: 상시로 설정하고 기간 설정한 페이지가 각각 있으면 뭐가 먼저 노출돼?
에이전트: 기간 설정 페이지가 우선입니다.

         - 기간공개 페이지 중 오늘 날짜가 기간 내인 것 → 최우선 노출
         - 기간 내인 페이지가 없으면 → 상시공개 페이지가 노출

         상시공개 페이지는 마스터(오피셜클럽)당 1개만 가능합니다.
         기간공개 페이지는 여러 개 만들 수 있지만, 기간이 겹치지 않아야 합니다.
```

---

## 검증 포인트 (eval 기준)

1. 기간공개 > 상시공개 우선순위 정확
2. 상시공개 = 마스터당 1개 제한 언급
3. 기간 겹침 불가 언급
