# #26 상품페이지만 비공개하고 메인/마스터 공개면 오류나?

## 분류
- 유저 의도: 도메인 질문 (explain_partial_visibility)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음

---

## Case A: 정상

```
유저: 상품페이지만 비공개하고 메인 페이지나 마스터 관리에서 공개 처리해놓으면 오류나? 어떻게 돼?
에이전트: 오류는 나지 않습니다.

         상품 페이지가 비공개(INACTIVE)이면, 오피셜클럽이나 메인 상품 페이지가
         공개 상태여도 해당 상품 페이지는 고객이 접근할 수 없습니다.
         URL로 직접 접근해도 "공개되지 않은 상품 페이지입니다"로 차단됩니다.

         참고로 히든 처리와 비공개는 다릅니다:
         - 히든: 목록에서만 숨김, URL 직접 접근은 가능
         - 비공개(INACTIVE): URL 직접 접근도 불가
```

---

## 검증 포인트 (eval 기준)

1. 오류 안 남 명확
2. INACTIVE면 URL 직접 접근도 차단 언급
3. 히든 ≠ 비공개 구분 안내
