# #6 어스플러스에 왜 상품페이지 노출이 안돼?

## 분류
- 유저 의도: 수행 (diagnose_visibility)
- 에이전트: 오케스트레이터 → 수행 에이전트 + 검증 에이전트

## 노출 체인 5단계
```
1. 마스터 PUBLIC
2. 메인 상품 ACTIVE
3. 페이지 ACTIVE
4. 기간 내 (상시 또는 startAt~endAt)
5. 옵션 isDisplay
(히든 페이지는 예외: 1,2 면제)
```

---

## Case A: 메인 상품 비활성

```
유저: 어스플러스에 왜 상품페이지 노출이 안돼?
에이전트: 어떤 오피셜클럽인가요?
유저: 봄날의 햇살
에이전트: (diagnose_visibility)
         📊 봄날의 햇살 노출 진단 결과

         ✅ 마스터 상태: PUBLIC
         ❌ 메인 상품 페이지: INACTIVE ← 원인
         ✅ 상품 페이지: 2개 ACTIVE

         🎯 원인: 메인 상품 페이지가 비활성화 상태입니다.
         구독 탭과 오피셜클럽 홈에서 마스터가 보이지 않습니다.
         메인 상품 페이지를 활성화하시겠어요?
```

## Case B: 마스터 비공개

```
유저: 왜 상품페이지가 안 보여?
에이전트: 어떤 오피셜클럽인가요?
유저: 김영익
에이전트: (diagnose_visibility)
         📊 김영익 노출 진단 결과

         ❌ 마스터 상태: PENDING ← 원인

         🎯 원인: 마스터가 아직 준비중(PENDING) 상태입니다.
         PUBLIC으로 전환해야 고객에게 노출됩니다.
         [오피셜클럽 관리로 이동 →] (/official-club/{cmsId})
```

## Case C: 기간 설정 문제

```
유저: 상품페이지 노출이 안돼
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (diagnose_visibility)
         📊 조조형우 노출 진단 결과

         ✅ 마스터 상태: PUBLIC
         ✅ 메인 상품 페이지: ACTIVE
         ✅ 상품 페이지 "월간 투자 리포트": ACTIVE
         ❌ 공개 기간: 2026-04-01 ~ 2026-04-30 (현재 기간 밖) ← 원인

         🎯 원인: 공개 기간이 아직 시작되지 않았습니다.
         기간을 변경하거나, 상시공개로 전환하시겠어요?
```

## Case D: 히든 페이지인 경우

```
유저: 상품페이지가 구독 탭에서 안 보여
에이전트: 어떤 오피셜클럽인가요?
유저: 봄날의 햇살
에이전트: (diagnose_visibility)
         📊 봄날의 햇살 노출 진단 결과

         상품 페이지 2개:
         - 단건 상품 (코드: 141) — 히든 🔒 (URL 직접 접근만 가능)
         - 단건상품 (코드: 124) — 공개 ✅

         💡 코드 141 페이지는 히든 처리되어 있어 구독 탭/오피셜클럽 홈에는 노출되지 않습니다.
         URL(/products/group/141)로만 접근 가능합니다.
         히든 해제하시겠어요?
```

---

## 검증 포인트 (eval 기준)

1. 노출 체인을 순서대로 확인하여 첫 번째 실패 지점 보고
2. 메인 INACTIVE인데 "노출 중"이라고 잘못 답변하면 안 됨 (Langfuse 버그 재발 방지)
3. 히든 페이지는 별도 안내 (URL 직접 접근 가능)
4. 기간 문제 시 현재 날짜와 설정 기간 명시
