# #5 상품이 왜 안나오지?

## 분류
- 유저 의도: 수행 (diagnose_product_display)
- 에이전트: 오케스트레이터 → 수행 에이전트
- 전제: 상품 페이지는 보이는 상태. 상품 옵션이 안 보이는 것.

## 사전조건 체인
```
오피셜클럽(cmsId) → 상품 페이지 특정 → 상품 옵션 isDisplay 체크
```

---

## Case A: 상품 옵션 비공개

```
유저: 상품이 왜 안나오지?
에이전트: 어떤 오피셜클럽의 상품인가요?
유저: 조조형우
에이전트: 어떤 상품 페이지의 상품인가요?
         1. 월간 투자 리포트 (코드: 148)
         2. 단건 특강 (코드: 155)
유저: 1번
에이전트: (get_product_list_by_page)
         상품 옵션 3개 확인했습니다:
         - 월간 구독 (29,900원) — 공개 ✅
         - 연간 구독 (299,000원) — 비공개 ❌
         - 프리미엄 (59,900원) — 비공개 ❌

         "연간 구독"과 "프리미엄"이 비공개(isDisplay: false) 상태입니다.
         공개로 변경하시겠어요?
```

## Case B: 전부 공개인데 안 보인다고 하면

```
유저: 상품이 왜 안나오지?
에이전트: 어떤 오피셜클럽의 상품인가요?
유저: 조조형우 월간 투자 리포트
에이전트: (get_product_list_by_page)
         상품 옵션 2개 모두 공개 상태입니다:
         - 월간 구독 (29,900원) — 공개 ✅
         - 연간 구독 (299,000원) — 공개 ✅

         상품 옵션은 정상입니다.
         혹시 상품 페이지 자체가 안 보이는 건가요?
         그렇다면 상품 페이지 노출 상태를 진단해드릴 수 있습니다.
```

---

## 검증 포인트 (eval 기준)

1. "상품" = 상품 옵션으로 이해 (상품 페이지가 아님)
2. 상품 옵션의 isDisplay 체크
3. 전부 공개인데 안 보인다면 → 상품 페이지 노출 진단으로 전환 제안
