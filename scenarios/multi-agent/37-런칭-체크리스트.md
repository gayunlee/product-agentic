# #37 새 마스터 상품 런칭하는데 빠진거 없는지 체크해줘

## 분류
- 유저 의도: 수행 (full_checklist)
- 에이전트: 오케스트레이터 → 수행 에이전트 + 검증 에이전트

## 체크 항목 (전체 사전조건 체인 + 노출 체인)
```
1. 마스터 그룹 존재
2. 오피셜클럽 존재 + PUBLIC
3. 시리즈 존재
4. 상품 페이지 존재 + ACTIVE
5. 상품 옵션 존재 + isDisplay
6. 메인 상품 페이지 ACTIVE
7. 이미지 등록 여부
```

---

## Case A: 모두 정상

```
유저: 새 마스터 상품 런칭하는데 빠진거 없는지 체크해줘
에이전트: 어떤 오피셜클럽인가요?
유저: 조조형우
에이전트: (전체 체인 검증)
         ✅ 오피셜클럽: 조조형우 (PUBLIC)
         ✅ 시리즈: 2개
         ✅ 상품 페이지: 월간 투자 리포트 (ACTIVE)
         ✅ 상품 옵션: 2개 (모두 공개)
         ✅ 메인 상품 페이지: ACTIVE
         ✅ 이미지: 대표 이미지 등록됨

         모든 항목이 정상입니다. 런칭 준비 완료!
```

## Case B: 일부 누락

```
유저: 새 마스터 상품 런칭하는데 빠진거 없는지 체크해줘
에이전트: 어떤 오피셜클럽인가요?
유저: 김영익
에이전트: (전체 체인 검증)
         ✅ 오피셜클럽: 김영익 (PUBLIC)
         ✅ 시리즈: 1개
         ✅ 상품 페이지: 경제 전망 리포트 (ACTIVE)
         ✅ 상품 옵션: 2개 (모두 공개)
         ❌ 메인 상품 페이지: INACTIVE
         ❌ 이미지: 플레이스홀더 이미지 (미등록)

         2개 항목이 미완료입니다:
         1. 메인 상품 페이지를 활성화해야 합니다. 활성화하시겠어요?
         2. 상품 페이지 이미지를 등록해주세요.
            [상품 페이지 수정으로 이동 →] (/product/page/{id}?tab=settings)
```

## 검증 포인트 (eval 기준)

1. 전체 사전조건 체인 + 노출 체인 순서대로 검증
2. 누락 항목마다 해결 방법 안내
3. 이미지 플레이스홀더 감지
