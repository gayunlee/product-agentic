# #12 시리즈가 필요하다는데 어디서 만들어야돼? 안 보이는데?

## 분류
- 유저 의도: 수행 (guide_create_series)
- 에이전트: 오케스트레이터 → 수행 에이전트
- API 호출: 없음 (안내만)

---

## Case A: 정상

```
유저: 시리즈가 필요하다는데 어디서 만들어야돼? 안 보이는데?
에이전트: 시리즈는 관리자센터가 아니라 파트너센터에서 생성합니다.
         관리자센터에서는 시리즈 조회만 가능합니다.
         [파트너센터로 이동 →] (master.us-insight.com)
         파트너센터 접속 후 [시리즈 만들기] 버튼을 클릭하시면 됩니다.
```

---

## 검증 포인트 (eval 기준)

1. 시리즈 생성 = 파트너센터 안내 (관리자센터 아님)
2. 파트너센터 URL 제공
