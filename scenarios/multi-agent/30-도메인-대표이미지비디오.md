# #30 대표이미지 대표비디오는 뭐야?

## 분류
- 유저 의도: 도메인 질문 (explain_main_content)
- 에이전트: 오케스트레이터 → 도메인 지식 에이전트
- API 호출: 없음

---

## Case A: 정상

```
유저: 이거 상품페이지에서 대표이미지 대표비디오는 뭐야?
에이전트: 상품 페이지 상단에 위치하는 대표 콘텐츠입니다.
         이미지 또는 동영상 중 1가지만 설정할 수 있습니다.

         - 이미지: JPG/PNG/WebP, 4MB 미만
         - 동영상: MP4, 30MB 이하
         - 이미지에는 클릭 시 이동할 URL 링크를 설정할 수 있습니다 (비디오는 불가)
```

---

## 검증 포인트 (eval 기준)

1. 이미지 또는 동영상 중 1가지만
2. 파일 형식/용량 제한 안내
3. URL 링크는 이미지만 가능
