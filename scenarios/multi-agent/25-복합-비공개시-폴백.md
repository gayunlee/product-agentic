# #25 노출중인 상품페이지 뭐가 있어? 비공개하면 다른거 공개되는거 있나?

## 분류
- 유저 의도: 복합 (query_pages + explain_fallback)
- 에이전트: 오케스트레이터 → 수행 에이전트 + 도메인 지식 에이전트

---

## Case A: 기간공개 비공개 → 상시공개 폴백

```
유저: 노출중인 조조형우 상품페이지 뭐가 있어? 비공개하면 다른거 공개되는거 있나?
에이전트: (목록 조회)
         현재 노출 중인 상품 페이지:
         - 월간 투자 리포트 (코드: 148, 기간공개: 3/1~3/31) ← 현재 노출 중

         다른 페이지:
         - 기본 구독 (코드: 130, 상시공개)

         월간 투자 리포트를 비공개하면, 상시공개인 "기본 구독"이 대신 노출됩니다.
```

## Case B: 상시공개 하나뿐

```
유저: 노출중인 봄날의 햇살 상품페이지 뭐가 있어? 비공개하면?
에이전트: (목록 조회)
         현재 노출 중인 상품 페이지:
         - 단건상품 (코드: 124, 상시공개)

         이 페이지를 비공개하면 노출되는 상품 페이지가 없어집니다.
         다른 기간공개 페이지 중 현재 날짜에 해당하는 것이 없습니다.
```

## Case C: 기간공개 여러 개

```
유저: 노출중인 조조형우 상품페이지 뭐가 있어?
에이전트: (목록 조회)
         현재 노출 중인 상품 페이지:
         - 봄 특강 (코드: 160, 기간공개: 3/1~4/30) ← 현재 노출 중

         다른 페이지:
         - 여름 특강 (코드: 161, 기간공개: 5/1~6/30) — 기간 밖
         - 기본 구독 (코드: 130, 상시공개)

         봄 특강을 비공개하면, 상시공개인 "기본 구독"이 대신 노출됩니다.
```

---

## 검증 포인트 (eval 기준)

1. 현재 노출 중인 페이지 정확히 특정
2. 비공개 시 폴백 규칙 안내: 기간공개 우선 → 없으면 상시공개 → 없으면 노출 없음
3. 현재 날짜 기준으로 기간 판단
