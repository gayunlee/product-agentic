# S002. 단건상품 세팅

## 메타데이터
- **카테고리**: happy_path
- **난이도**: hard
- **필요 권한**: PRODUCT + MASTER
- **관련 Phase**: 0 → 1 → 2 → 3 → 4 → 5

## 사용자 입력
> "김영익 마스터로 단건상품 만들어줘. 투자 특강, 60일 이용, 99000원"

## 기대 행동

### Phase 0: 마스터/시리즈 확인
1. 마스터 검색
   - Tool: `search_masters(search_keyword="김영익")`
   - 기대 결과: cmsId, publicType "PUBLIC" 확인
2. 시리즈 확인
   - Tool: `get_series_list(master_id="{cmsId}")`
   - 기대 결과: 시리즈 확인, contentIds 확보

### Phase 2: 상품 페이지 생성
3. 상품 페이지 생성 (**단건 차이점: isChangeable=false**)
   - Tool: `create_product_page(master_id="{cmsId}", title="투자 특강 페이지", product_type="ONE_TIME_PURCHASE", is_changeable=False, is_always_public=True, is_always_apply=True)`
   - 기대 결과: 페이지 생성 성공
4. 상품 페이지 목록 조회
   - Tool: `get_product_page_list(master_id="{cmsId}")`

### Phase 3: 상품 옵션 등록
5. 상품 옵션 생성 (**단건 차이점: paymentPeriod 대신 usagePeriodType + useDuration**)
   - Tool: `create_product(product_group_id="{pageId}", name="투자 특강", origin_price=99000, content_ids=["{seriesId}"], display_texts=["투자 특강 시리즈 이용권"], usage_period_type="DURATION", use_duration=60)`
   - 기대 결과: 상품 생성 성공

### Phase 4: 활성화
6~9. 활성화 4단계 (S001과 동일)

### Phase 5: 검증
10~11. 검증 (S001과 동일 구조)

## 기대 응답
- 마스터명: 김영익
- 상품 타입: 단건 (60일 이용)
- 금액: 99,000원 (1회)
- 검증 결과 + 관리자센터 링크 + 체크리스트

## 절대 하면 안 되는 것
- paymentPeriod 사용 (단건에는 없음)
- isChangeable을 true로 설정 (단건은 반드시 false)
- 구독 회차 한정 할인(SUBSCRIPTION_COUNT_DISCOUNT) 사용 (단건은 상시 할인만 가능)
- 시리즈 확인 없이 상품 생성

## 평가 기준
- [ ] product_type "ONE_TIME_PURCHASE" 정확히 지정
- [ ] is_changeable=False (단건 필수)
- [ ] usage_period_type "DURATION" + use_duration 60 지정
- [ ] paymentPeriod 미사용
- [ ] 금액 99000 정확히 전달
- [ ] 활성화 4단계 모두 수행
