# S001. 구독상품 전체 세팅

## 메타데이터
- **카테고리**: happy_path
- **난이도**: hard
- **필요 권한**: PRODUCT + MASTER
- **관련 Phase**: 0 → 1 → 2 → 3 → 4 → 5

## 사용자 입력
> "조조형우 마스터로 구독상품 세팅해줘. 월간 투자 리포트, 월 29900원"

## 기대 행동

### Phase 0: 마스터/시리즈 확인
1. 마스터 검색
   - Tool: `search_masters(search_keyword="조조형우")`
   - 기대 결과: cmsId "35", name "조조형우", publicType "PUBLIC" 확인
2. 시리즈 확인
   - Tool: `get_series_list(master_id="35")`
   - 기대 결과: 시리즈 1개 이상 확인, contentIds 확보

### Phase 2: 상품 페이지 생성
3. 상품 페이지 생성
   - Tool: `create_product_page(master_id="35", title="월간 투자 리포트 페이지", product_type="SUBSCRIPTION", is_always_public=True, is_always_apply=True)`
   - 기대 결과: 페이지 생성 성공 (INACTIVE 상태로 시작)
4. 상품 페이지 목록 조회 (생성 확인)
   - Tool: `get_product_page_list(master_id="35")`
   - 기대 결과: 방금 생성한 페이지 확인, productGroupId 확보

### Phase 3: 상품 옵션 등록
5. 상품 옵션 생성
   - Tool: `create_product(product_group_id="{pageId}", name="월간 투자 리포트", origin_price=29900, content_ids=["{seriesId}"], display_texts=["투자 리포트 시리즈 이용권"], payment_period="ONE_MONTH")`
   - 기대 결과: 상품 생성 성공, productId 확보

### Phase 4: 활성화
6. 상품 노출 ON
   - Tool: `update_product_display(product_id="{productId}", is_display=True)`
7. 상품 순서 설정
   - Tool: `update_product_sequence(product_group_id="{pageId}", product_ids=["{productId}"])`
8. 상품 페이지 공개
   - Tool: `update_product_page_status(product_page_id="{pageId}", status="ACTIVE")`
9. 메인 상품 페이지 활성화
   - Tool: `update_main_product_setting(master_id="35", view_status="ACTIVE")`

### Phase 5: 검증
10. 세팅 결과 확인
    - Tool: `get_product_page_detail(product_page_id="{pageId}")`
    - Tool: `get_product_list_by_page(product_page_id="{pageId}")`
11. 탐색 에이전트에 검증 요청
    - Tool: `verify_product_setup(product_group_code={code}, master_id="35", master_display_index={idx}, master_name="조조형우", product_name="월간 투자 리포트", price=29900, product_count=1)`

## 기대 응답
에이전트가 최종적으로 사용자에게 보여줄 핵심 포인트:
- 마스터명: 조조형우
- 상품 페이지명 + 코드
- 등록된 옵션: 월간 투자 리포트 29,900원/월
- 검증 결과 (PASS/FAIL)
- 관리자센터 링크
- 고객 화면 링크
- 공개 전 체크리스트 (이미지 교체, 유의사항 등록 등)

## 절대 하면 안 되는 것
- 시리즈 확인 없이 상품 옵션 생성 (contentIds 빈 배열 불가)
- 마스터 검색 없이 상품 페이지 생성 (masterId 필수)
- 상품 페이지 없이 상품 옵션 등록 (productGroupId 필수)
- 상품 삭제 Tool 호출
- 마스터의 id를 masterId로 사용 (반드시 cmsId 사용)

## 평가 기준
- [ ] Phase 순서 준수 (0→2→3→4→5)
- [ ] 시리즈 확인 후 상품 생성 (선행조건 준수)
- [ ] masterId에 cmsId 사용 (id 아님)
- [ ] 상품 타입 SUBSCRIPTION 정확히 지정
- [ ] paymentPeriod ONE_MONTH 지정
- [ ] 금액 29900 정확히 전달
- [ ] 활성화 4단계 모두 수행
- [ ] 최종 검증 조회 수행
- [ ] 체크리스트 안내 포함
