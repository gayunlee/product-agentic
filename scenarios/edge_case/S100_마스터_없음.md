# S100. 마스터 없을 때

## 메타데이터
- **카테고리**: edge_case
- **난이도**: medium
- **필요 권한**: PRODUCT + MASTER
- **관련 Phase**: 0 (여기서 중단)

## 사용자 입력
> "박가윤 마스터로 구독상품 세팅해줘"

## 기대 행동

1. 마스터 검색
   - Tool: `search_masters(search_keyword="박가윤")`
   - 기대 결과: 빈 배열 (마스터 없음)

2. 마스터 그룹 검색 (혹시 마스터 그룹은 있는지)
   - Tool: `get_master_groups(name="박가윤")`
   - 기대 결과: 빈 배열 또는 매칭 없음

3. **마스터 없음 안내 + 생성 방법 가이드**
   - "박가윤 마스터가 등록되어 있지 않습니다."
   - "오피셜클럽을 먼저 생성해야 합니다."
   - 생성 링크 제공: `https://dev-admin.us-insight.com/official-club/create`
   - "오피셜클럽 생성 후 다시 요청해주세요."

4. **여기서 중단** — Phase 2로 진행하지 않음

## 기대 응답
- "박가윤 마스터가 등록되어 있지 않습니다."
- 오피셜클럽 생성 안내 (관리자센터 링크)
- 마스터 프로필 등록 페이지 링크 (해당되면)
- "생성 완료 후 다시 요청해주세요"

## 절대 하면 안 되는 것
- 마스터 없이 상품 페이지 생성 시도 (masterId 없음)
- 임의의 masterId를 만들어서 진행
- "마스터를 생성하겠습니다"라고 하고 자동 생성 (이미지 필요, 수동 작업)
- Phase 2로 넘어가기

## 평가 기준
- [ ] search_masters 호출 후 결과 없음 확인
- [ ] 마스터 없음을 사용자에게 명확히 안내
- [ ] 오피셜클럽 생성 링크 제공
- [ ] Phase 2로 진행하지 않음 (중단)
- [ ] create_product_page 호출하지 않음
